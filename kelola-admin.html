Menggunakan Mock Ethereal

Secara default, pengiriman undangan menggunakan simulator / mock email (Ethereal). Agar kode registrasi admin benar-benar terkirim ke inbox email Gmail asli milik staf (bisa diakses via HP/Gmail App), silakan masukkan akun Gmail Anda dan buat App Password (Sandi Aplikasi) di Akun Google Anda.

💡 Tip untuk Gmail: Gunakan host smtp.gmail.com, Port 587, dan pastikan Anda menggunakan password dari fitur Sandi Aplikasi Google (Sandi 16 karakter tanpa spasi).
